Get-there-bot

A bot for Twitch.tv that lets card game players know the odds.

Download this project as a .zip file Download this project as a tar.gz file

Using Get There Bot

Command Example Description
!calculate_outs [cards] [outs] [draws] !calculate_outs 53 9 3 Calculate the chance to draw one of [outs] outs in the next [draws] draws with [cards] cards left in your deck.

All of these commands accept double hypenated options named the same as their positional arguments (these can be in any order).

Ex: !calculate_outs 53 --outs 6 --draws 3

Getting Get There Bot

Make sure you are logged in on the account whose channel you want GetThereBot to join.

Go to Get There Bot's channel and type !join and GetThereBot will join your channel.

You can later type !leave and GetThereBot will leave your channel.
